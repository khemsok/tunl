tunl

enter the tunnel. block everything out.

$ bun add -g @khemsok/tunl 

      

    

  







  


    
    

      
what it does


      

        

          01
          

            
blocks distractions

            
twitter, reddit, youtube — blocked while you focus. customize your blocklist anytime.

          

        

        

          02
          

            
reveals animated art

            
scenes build in real-time. stars twinkle, windows flicker, buildings rise. multiple themes.

          

        

        

          03
          

            
tracks your streaks

            
sessions, total focus time, day streaks. build the habit. check with tunl --stats

          

        

        

          04
          

            
zero config

            
first run walks you through setup. after that, just type tunl.

          

        

      

    


    
    

      
controls


      

        

          space
          start / pause / resume
        

        

          r
          stop session
        

        

          q
          quit
        

        

          +/-
          adjust timer
        

        

          s
          edit blocked sites
        

        

          t
          change theme
        

        

          i
          view stats
